# План тестирования Virage 3D Editor

Этот документ описывает ручные тесты, которые необходимо проходить после каждого значительного изменения в коде. Каждый тест имеет чёткие шаги и ожидаемый результат.

---

## 1. Запуск и стартовая сцена

| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1.1 | Открыть `http://localhost:3000/`, дождаться загрузки | Страница загружается без ошибок в консоли |
| 1.2 | Проверить стартовую сцену | На сцене три фигуры: куб (слева), сфера (центр), цилиндр (справа) |
| 1.3 | Проверить панель инструментов | Видны кнопки: Virage 3D, 3 булевых, Удалить, Очистить всё, На стол, Сохранить, Загрузить, Экспорт STL, Импорт STL, Отмена, Повтор |
| 1.4 | Проверить статус-бар | Видны: Перемещение/Вращение/Масштаб, сетка 5мм, кнопка Стол, статус, ссылки автора |
| 1.5 | Проверить панель свойств | Текст «Выберите объект для редактирования» |
| 1.6 | Проверить левую панель «Фигуры» | Заголовок «Фигуры», 5 кнопок: Куб, Сфера, Цилиндр, Конус, Тор |
| 1.7 | Проверить статус-текст | Текст «Готово — Virage 3D Editor» |
